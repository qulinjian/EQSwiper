EQSwiper V 1.0.0

一款不需要依赖任何库的轮播图插件

如果您的网页需要这样的轮播效果，您可以考虑使用EQSwiper。

github地址：https://github.com/qulinjian/EQSwiper

EQSwiper用法很简单，上个例子中，您只需把结构写成这样：

			<div id="mySwiper1">
	<div style="background: #ed5a65;height: 200px;"></div>
	<div style="background: #1177b0;height: 200px;"></div>
	<div style="background: #40a070;height: 200px;"></div>
	<div style="background: #fbc82f;height: 200px;"></div>
</div>

引入EQSwiper.js之后，想让它动起来，只要这样做：

			new EQSwiper({
	el: "#mySwiper1"
})

是的，只需定义好选择器就可以做出一个带有动画效果，能自动播放的轮播图啦！

EQSwiper的体积很小，只有不到7K，代码只有200行。但可以应用于绝大多数的轮播图效果。

请注意，如果您的轮播图需要兼容IE或低版本的谷歌，低版本的火狐浏览器，EQSwiper不是最好的选择，您可以选择那些功能更齐全的插件，他们会把兼容性处理的很好。如果您打算将您的轮播图用于移动端，请慎用。1.0版本的EQSwiper兼容并不完美。目前主要的应用场景是PC端高版本浏览器。我会在下一个版本中对移动端进行改进。

如果对轮播图有其他需求，您可以在配置项中修改对应的配置参数。配置参数共分成4类，分别用于控制整个轮播图、轮播图动画效果、两侧箭头以及下侧圆点。参数之间互相没影响，可以放心使用。

举个例子，如果想把轮播图做成一个初始展示第2张幻灯片，两秒中自动切换到下一张，并且把左右箭头设置成不同颜色，同时把下面的小圆点变成正方形按钮，像这样：

您可以这样设置：

			new EQSwiper({
	el: "#mySwiper2",
	global: {
		initialSlide: 1
	},
	play: {
		delay: 2
	},
	arrow: {
		cssLeft: "border-color: #322f3b",
		cssRight: "border-color: #9fa39a"
	},
	round: {
		cssNormal: "border-radius: 0",
		cssActive: "border-radius: 0"
	}
})

上面例子中涵盖了EQSwiper的全部配置大项。除了el为必选配置项之外，其他配置项均是可选的。

以下是全部配置参数说明：

一、global

参数名 类型 默认值 描述
width 数字、百分比、auto auto swiper的整体宽度
height 数字、百分比、auto auto swiper的整体高度
initialSlide 数字 0 默认显示第几张幻灯片
callback 函数 - 每次切换完成之后执行(函数的第一个参数为当前幻灯片的索引)

二、play

参数名 类型 默认值 描述
animate true / false true 是否开启动画效果
autoPlay true / false true 是否自动播放
speed 数字 1 运动持续时长（单位：秒）
delay 数字 8 每张幻灯片停留时长（单位：秒）

三、arrow

参数名 类型 默认值 描述
show true / false true 是否开启两侧箭头
color 颜色值 rgba(0,0,0,.4) 箭头的颜色
size 数字 20 箭头的大小
offset 数字 30 箭头距离左右两侧的位置
cssLeft 字符串 空字符串 左侧箭头的样式
cssRight 字符串 空字符串 右侧箭头的样式

四、round

参数名 类型 默认值 描述
show true / false true 是否开启下面小圆点
size 数字 14 圆点大小
offset 数字 18 圆点距离下面的偏移量
color 颜色值 rgba(0,0,0,.4) 默认圆点的颜色
activeColor 颜色值 rgb(255,255,255) 当前幻灯片对应圆点的颜色
cssNormal 字符串 空字符串 默认圆点的样式
cssActive 字符串 空字符串 当前幻灯片对应圆点的样式
